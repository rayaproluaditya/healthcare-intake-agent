# Healthcare Intake Agent

An AI-powered healthcare intake system using Groq's fast inference API for preliminary symptom analysis and structured patient data collection.

## Features

- 🏥 **Clinical Intake Agent**: Professional, empathetic conversation flow
- 🔍 **RAG Pipeline**: Retrieval-augmented generation with medical guidelines
- 🛡️ **Safety Guardrails**: Emergency detection and safety compliance
- 📊 **Structured Data**: Extracts and formats patient information
- 🚀 **Fast Inference**: Powered by Groq's high-performance API
- 🐳 **Dockerized**: Easy deployment with Docker Compose
